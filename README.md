A web-based Department Mailing System developed using ASP.NET Web Forms and C#. This application helps educational institutions manage department announcements, events, student notifications, and feedback through a centralized platform.

## 🚀 Features

- 👨‍💼 Admin Login
- 📩 Department Mail Management
- 📢 Event Management
- 👨‍🎓 Student Feedback Management
- 📚 Student Semester Update
- 📄 View Mail Details
- 🗄️ SQL Server Database Integration

## 🛠️ Tech Stack

- Frontend: ASP.NET Web Forms (ASPX)
- Backend: C#
- Database: SQL Server (.mdf)
- IDE: Microsoft Visual Studio
- Framework: .NET Framework

## ⚙️ Installation

1. Clone the repository.

```bash
git clone https://github.com/your-username/DepartmentMailingProject.git
```

2. Open the project in Visual Studio.

3. Restore the SQL Server database from the **App_Data** folder.

4. Configure the connection string if required.

5. Build and run the project using IIS Express.

## 🎯 Project Objectives

- Simplify communication between departments and students.
- Manage departmental announcements efficiently.
- Maintain student feedback records.
- Reduce manual notification processes.
